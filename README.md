## 📌 Assignment 1: Library System

A simple Prolog-based system that models:

* Students and books
* Borrowing records
* Book ratings
* Topics per book

### 🔹 Features

* List processing (member, append, length)
* Find borrowed books per student
* Count borrowers for each book
* Get most borrowed book
* Find top reviewer
* Analyze book topics
